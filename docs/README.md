##구현 기능 목록

1. 역(Station)
2. 노선(Line)
    - 이름
    - 자신에게 포함되어 있는 구간을 List로 갖고 있다.
3. 구간(Section)
    - 역과 역 사이의 시간 (Time)
    - 역과 역 사이의 거리 (Distance)
    - 2개의 역을 갖고 있다.
4. 화면(Screen) 생성
    - 메인화면
5. 사용자 입력값을 받는 기능(InputView)

6. 결과를 출력하는 기능(OutputView)

6. 탐색할 역을 입력받는 기능

7. 입력받은 역까지의 이동 시간을 구하는 기능

8. 입력받은 역까지의 이동 거리를 구하는 기능

9. 결과를 출력하는 기능
